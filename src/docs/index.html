****BOOKING API****

Welcome to the booking API! With this API you can:

Update bookings
Update Hosts and Users
Update properties

The booking and user-update parts are protected with jwt-token and optional Auth0

Furthermore database connections are now established with Planetscale and Prisma

Try some of the GET requests endpoints:
amenities
bookings
properties
users
hosts

To make Create, Update, and Delete you need a jwt-Token. With POST login and password to get your jwt-Token. It is not accessible through GET request. Try to open and copy the link in postman as a POST request with header {"username": "asimpson", "password": "password234"} : login
